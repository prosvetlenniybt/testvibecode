1. Появление элементов (Enter)

Как мягко вводить новые блоки в интерфейс: карточки, уведомления, модальные окна.

Fade In — простое появление
opacity

Кейс: список карточек, которые появляются по мере загрузки данных.

  • Хорошо подходит для второстепенных элементов.
  • Не отвлекает пользователя от основного фокуса.
Промпт для GPT
Сделай карточку, которая плавно появляется (fade in) при нажатии на кнопку. Используй CSS transition по opacity и transform, без сторонних библиотек. HTML/CSS/JS в одном файле.

2. Исчезновение элементов (Exit)

Как аккуратно убирать элементы из интерфейса, не ломая контекст пользователя.

Fade Out — плавное исчезновение
opacity

Кейс: уведомление, которое исчезает через несколько секунд.

  • Можно скрывать по таймеру или по клику.
  • Хорошо подходит для success/error сообщений.
Промпт для GPT
Сделай toast-уведомление, которое исчезает (fade out) при нажатии на кнопку «Скрыть». Используй CSS transition по opacity и лёгкий сдвиг по X/Y.

3. Анимация экранов

Как анимировать переходы между экранами без сложных роутеров и фреймворков.

Swipe между экранами
translateX

Кейс: мобильное приложение или мобильная версия сайта.

  • Горизонтальный свайп с индикатором текущего экрана.

4. Взаимодействия

Как сделать клики, hover и ошибки более «живыми».

Button Press — отзывчивая кнопка
scale + shadow

Кейс: primary-кнопка, которая должна чувствоваться «живой».

  • Лёгкое нажатие вниз по клику.
  • Сильнее подсвеченный hover-состояние.
Промпт для GPT
Сделай кнопку с эффектом «нажатия»: — при наведении поднимается и усиливает тень, — при клике слегка уменьшается и опускается. Используй transform и box-shadow.
DEMO · BUTTON PRESS

Hover Lift — поднятие карточки
hover

Кейс: сетка карточек, где hover подчёркивает доступность клика.

  • Небольшой подъём и усиление тени.
  • Лёгкое изменение фона и рамки.
Промпт для GPT
Сделай сетку карточек, которые при наведении слегка приподнимаются, усиливают тень и меняют цвет рамки. Без JS, только через :hover.

Shake — ошибка ввода
keyframes

Кейс: неправильный пароль, пустое обязательное поле.

  • Краткая тряска по оси X.
  • Хорошо работает в паре с красным цветом и подсказкой.
Промпт для GPT
Сделай поле ввода с кнопкой «Войти». Если пользователь ввёл пустое поле и нажал кнопку — блок с ошибкой трясётся. Анимация через @keyframes.

6. Микро-анимации и геймификация

Маленькие эффекты, которые добавляют удовольствия и «сочности» в UI.

Ripple — волна по нажатию
ripple

Кейс: кнопки в стиле Material Design.

  • Волна расходится от точки клика.
  • Хорошо работает на контрастных кнопках.
Промпт для GPT
Сделай кнопку с ripple-эффектом: при клике из точки нажатия расходится волна. Без библиотек, через JS и CSS.

Success Pop — всплывающий успех

  • Небольшой pop-in эффект с bounce.
  • Можно использовать для галочек, трофеев, бейджей.

7. Reward & Confetti

Как дополнительно «подсветить» достижение: анимации награды и конфетти.

Reward Confetti
confetti

Кейс: достижение уровня, завершение курса, закрытие сложного задания.

  • Конфетти падают сверху в зоне награды.
  • Можно комбинировать с текстом «Поздравляем!» и попапом.
Промпт для GPT
Сделай зону награды с кнопкой «Получить награду». При клике по кнопке запускается эффект конфетти и появляется надпись «Награда получена».
