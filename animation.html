UI Motion Patterns

Методичка по анимациям: название · описание · живой пример. Все примеры сделаны на чистых HTML/CSS/JS и подходят для статических сайтов и GitHub Pages.

1. Исчезновение и Появление (БАЗА)

Анимации, отвечающие за открытие/скрытие блоков: закрытие баннеров, модалок, карточек, состояний ошибки.

Fade Out
Exit

Элемент плавно исчезает без смещения. Подходит для мгновенного скрытия подсказок, чипов, всплывающих сообщений.

Пример промпта для GPT: «Сделай анимацию исчезновения уведомления: прозрачность от 1 до 0 за 250ms, easing: ease-in».

Параметры
duration: 250ms easing: ease-in

Заметка по easing: появление — ease-out, исчезновение — ease-in.

Пример
Fade Out
Slide + Fade Exit
Exit

Элемент чуть поднимается и исчезает. Хорошо смотрится для заметок, баннеров, временных карточек.

Пример промпта для GPT: «Сделай анимацию закрытия баннера: подними блок на 18px и уменьшай прозрачность до 0 за 280ms с easing ease-in».

Параметры
duration: 280ms easing: ease-in

Пример
Slide Exit

2. Переходы между экранами

Паттерны для онбордингов, пошаговых мастеров, переключения между псевдо-страницами и вкладками.

Screen Swipe (Left / Right)
Screen transition

Два «окна» внутри области сменяют друг друга горизонтальным свайпом.

Пример промпта для GPT: «Сделай два шага онбординга, которые переключаются горизонтальным свайпом: анимация translateX за 400ms ease-in-out».

Параметры
duration: 400ms easing: ease-in-out

Кнопки «Назад»/«Вперёд» двигают трек с окнами влево и вправо.

Пример (2 шага онбординга)

3. Анимации взаимодействий

Rклики, наведение и ошибки.

Button Press
Interaction

Лёгкое «сжатие» кнопки при клике. Самый базовый паттерн для любых интерактивных кнопок.

Пример промпта для GPT: «Сделай кнопку, которая при клике слегка уменьшается (scale 0.94) и опускается на 1px за 120–150ms с easing ease-out».

Параметры
duration: 120–150ms easing: ease-out

Пример (нажми на кнопку) :active state
Hover Lift
Interaction

Карточка слегка поднимается и отбрасывает тень при наведении.

Пример промпта для GPT: «Сделай карточку, которая при наведении поднимается на 3px, усиливает тень и меняет цвет рамки за 180ms с easing ease-out».

Параметры
duration: 180ms easing: ease-out

Пример (наведи курсор) hover
Наведи на карточку — она «поднимется».
Shake (ошибка)
Interaction

Лёгкое «поддёргивание» поля или блока при ошибке.

Пример промпта для GPT: «Сделай анимацию ошибки для поля ввода: лёгкий shake по оси X за 300–350ms с easing ease-in-out и красной рамкой».

Параметры
duration: 300–350ms easing: ease-in-out

Идеально для форм: поле с ошибкой слегка трясётся и подсвечивается красным.

Пример
Ошибка ввода

4. Появление при скролле

Элементы появляются по мере прокрутки, создавая ощущение глубины и последовательности.

Reveal on Scroll
Scroll

Элементы по очереди проявляются снизу при попадании в viewport. Подходит для списков преимуществ, шагов, таймлайнов.

Пример промпта для GPT: «Сделай список блоков, которые при попадании во viewport проявляются и поднимаются вверх за 500ms ease-out, используя IntersectionObserver».

Параметры
duration: 500ms easing: ease-out

5. Микро-анимации

Ripple
Micro

Волна расходится от точки клика.

Пример промпта для GPT: «Сделай кнопку с ripple-эффектом клика: круг увеличивается и исчезает за 600ms с линейной анимацией».

Параметры
duration: ~600ms easing: linear

Работает через создание span-элемента в точке клика и анимацию его масштаба и прозрачности.

Пример (кликни по кнопке) JS + keyframes
Success Pop
Micro

Иконка успеха появляется с лёгким увеличением. Подходит для подтверждения действий: сохранено, отправлено, завершено.

Пример промпта для GPT: «Сделай иконку успеха, которая появляется с небольшим pop-эффектом (scale от 0.7 до 1) за 300ms ease-out и меняет текст рядом на “Готово”».

Параметры
duration: 300ms easing: ease-out

Можно сочетать с изменением текста кнопки или появлением небольшого тултипа «Готово».

Пример
✓

6. Геймификационные анимации

Анимации награды, прогресса

Reward Confetti
Gamification

Короткий всплеск конфетти и пульсация блока. Используется при завершении онбординга, челенджа или сложного действия.

Пример промпта для GPT: «Сделай блок награды с кнопкой: при нажатии запускается короткий всплеск конфетти вокруг блока и лёгкий scale-пульсация за ~800–900ms».

Параметры
duration: ~800–900ms easing: ease-out

Конфетти разлетается, блок подтверждения слегка деформируется за счёт scale-анимации.

Пример
Успех! Награда получена
XP Progress Bar
Gamification

Полоса опыта, заполняющаяся по мере выполнения заданий. Классический игровой паттерн, который отлично ложится на обучение и чек-листы.

Пример промпта для GPT: «Сделай XP-полосу, которая при каждом клике по кнопке увеличивает ширину прогресса на 10% за 600ms с cubic-bezier(.19,1,.22,1) и обновляет текст “N XP”».

Параметры
duration: 600ms easing: cubic-bezier(.19,1,.22,1)

Можно обновлять XP после каждого значимого действия пользователя и визуально показывать приближение к следующему уровню.

Пример
Прогресс опыта 0 XP

Gamification

Плашка «+50 XP» всплывает над карточкой задания и исчезает.

Пример промпта для GPT: «Сделай всплывающую плашку “+50 XP”: она появляется по центру карточки, всплывает вверх и исчезает за ~800ms ease-out».

Параметры
duration: ~800ms easing: ease-out

Level Up
Gamification

Иконка "подсвечивается", а уровень растёт. Чистый игровой паттерн для статусов и рангов.

Пример промпта для GPT: «Сделай анимацию Level Up для иконки: круг масштабируется с лёгким pop-эффектом за 400ms ease-out, поверх включается сияние, счётчик уровня увеличивается».

Параметры
duration: ~400ms easing: ease-out

Можно запускать при повышении уровня, завершении серии заданий или достижении streak.
